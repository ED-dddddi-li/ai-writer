# ✍️ AI 写作助手

> 输入主题，一键生成公众号文章、小红书笔记、知乎回答、朋友圈文案。

**🔗 在线体验：[https://ed-dddddi-li.github.io/ai-writer/](https://ed-dddddi-li.github.io/ai-writer/)**

**技术栈**：HTML · CSS · JavaScript · DeepSeek API

---

## ✨ 功能

- 📱 **公众号**：支持多种风格（干货/故事/深度/幽默），800-2500字可调，自动生成标题备选+话题标签
- 📕 **小红书**：emoji 风格笔记，干货/种草/经验/避坑四种类型
- 🔵 **知乎**：高赞回答风格，多种答题角度，结构清晰有说服力
- 💬 **朋友圈**：一次生成多条备选，风格各异随意挑选

---

## 🚀 使用方法

1. 打开[在线地址](https://ed-dddddi-li.github.io/ai-writer/)
2. 填入 DeepSeek API Key
3. 选择平台标签，填写主题和风格
4. 点击生成，一键复制结果

> Key 在本地使用，不会上传至任何服务器

---

## 🔑 获取 API Key

在 [platform.deepseek.com](https://platform.deepseek.com) 注册即可获取，新用户有免费额度。
